# Sales Analysis by Region using Google Sheets

## Tools Used
- Google Sheets

## Analysis Done
- Sales by Region

## Key Insights
- West region contributes the highest share of total sales (~31%)
- East region follows closely (~29%), indicating strong performance
- Central region shows moderate contribution (~22%)
- South region has the lowest share (~17%), indicating potential growth opportunity

## Conclusion
The analysis shows that sales are concentrated in specific regions, with West and East leading. This insight can help businesses focus on underperforming regions like South to improve overall performance.
